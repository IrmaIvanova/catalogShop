# Online Courses Catalog

Небольшой карточный каталог онлайн-курсов с поиском и фильтрацией, реализованный в качестве тестового задания на вакансию Frontend-разработчик (React / Next.js).  
Ключевые аспекты проекта: адаптив, работа с состоянием, чистая архитектура и vanilla JavaScript.
